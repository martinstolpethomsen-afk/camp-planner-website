FAQ

Questions before launch.

Can I start without paying?

Yes. Organizers can start free, test the event flow and then scale when ready.

Is the €2 participant fee final?

It is the current launch positioning and may be adjusted based on pilot feedback and event size.

Can federations run multiple events?

Yes. Organization pricing is intended for multi-event structures and shared administration.

Does Camp-Planner handle payments?

The platform is being prepared for registration, finance overview and integration-ready workflows. Payment handling depends on launch setup and local requirements.
